# 3D Model Viewer
A lightweight, browser-based 3D model viewer built with Three.js, HTML, and Bootstrap 5 for styling.
Created as a personal tool for quickly inspecting and converting 3D assets without needing to open a full DCC application like Blender or Maya.

# Features
- Load and preview 3D models directly in the browser, no install required
- Import support for **GLTF, GLB, OBJ, FBX, STL, PLY and 3MF**
- Export models to **GLB, GLTF, OBJ, STL and PLY**
- Adjustable lighting; directional light colour, directional intensity and ambient intensity
- Wireframe and grid floor toggles
- Scene stats; live mesh, vertex and triangle counts
- Light/Dark theme toggle
